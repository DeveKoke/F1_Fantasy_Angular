FANTASY F1

This is a demo application (desktop version) fora Fantasy F1 made with ANGULAR framework.

You might be wondering what a Fantasy is. It's a real game where you, as a user, select a team made up of current F1 drivers. To build that team, you'll need to consider your budget and the market value of the drivers.

Once you have your team assembled, points will be assigned based on the actual performance of your chosen drivers in the weekend's real Grand Prix.

This demo was built to practice handling Angular 16 along with TypeScript and Sass for styling. In doing so, I was able to practice event binding, component communication, the use of interfaces and services, and setting up routing among others. An API has been used to obtain information about circuits and drivers.
